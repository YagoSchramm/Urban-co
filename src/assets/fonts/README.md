Put the local font files in this folder with these exact names:

- `Montserrat-Regular.woff2` or `Montserrat-Regular.ttf`
- `MontserratAlternates-Regular.woff2` or `MontserratAlternates-Regular.ttf`
- `MontserratAlternates-SemiBold.woff2` or `MontserratAlternates-SemiBold.ttf`
- `MontserratAlternates-Bold.woff2` or `MontserratAlternates-Bold.ttf`

You can use `woff2`, `ttf`, or both.
If both exist, the browser will prefer `woff2`.
